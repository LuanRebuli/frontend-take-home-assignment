## üíª Getting started


**Clone the project and access the folder**

```bash
$ git clone https://github.com/rodolfomori/frontend-take-home-assignment && cd frontend-take-home-assignment
```

**Install dependencies**

```bash
$ npm install
```

**Follow the steps below**

```bash

# Start the client
$ npm run dev
```
